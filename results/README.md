# Results

## Behavioral comparison

- Moderate-noise GRU: \(d' = 2.49\), criterion \(c = 0.41\)
- Mouse session 1119946360: \(d' = 1.68\), criterion \(c = 0.56\)
- At a matched mouse false-alarm rate of 6.7%, the RNN achieved 81.8% hits,
  compared with 61.1% in the mouse.

## VISp neural results

- Units: 106 good, quality-controlled VISp neurons
- Trials: 135 hits and 86 misses
- Hit-minus-miss evoked response: +1.60 Hz in the 0–500 ms post-change window
- Paired Wilcoxon test: \(p = 2.45 \times 10^{-9}\)
- Population hit/miss decoding: balanced accuracy = 74.4%, ROC-AUC = 0.810
- Permutation test: \(p = 0.005\), 200 permutations

## Scope

All mouse neural conclusions are within-session results from one ecephys session.
They show association rather than causal influence.
