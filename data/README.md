# Data

This repository does not redistribute the Allen Institute NWB session file.

## Source data

Mouse electrophysiology and behavioral data were obtained from the Allen Institute
Visual Behavior Neuropixels dataset.

- Ecephys session ID: `1119946360`
- Primary region analyzed: primary visual cortex (`VISp`)
- Raw NWB file: `ecephys_session_1119946360.nwb` (~3.4 GB)

Download the raw session through the AllenSDK or the official Allen data resources,
then set the NWB path in the analysis notebooks.

## Included derived data

The `derived/` directory contains lightweight tables generated by the notebooks,
including trial outcomes, unit quality-control metadata, spike-count summaries,
PETH summaries, and decoder/permutation results.
